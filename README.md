# 桌面宠物 App

基于 Electron + HTML/CSS/JavaScript 的 Windows 桌面宠物应用。它使用透明无边框悬浮窗口展示 GIF / WebP / 视频素材，并提供控制面板管理素材、互动按钮和随机文案。

## 功能

- 透明无边框悬浮窗口，始终置顶展示桌宠。
- 首次启动显示简约默认宠物形象，并提示前往控制面板上传素材。
- 支持鼠标拖动桌宠位置，位置会自动保存。
- 单击桌宠触发随机回应文案。
- 悬停桌宠自动显示互动按钮和“面板”入口。
- 控制面板支持上传、切换、删除素材。
- 支持 `webp`、`webm`、`mp4`、`mov`、`gif` 文件。
- 支持自定义互动按钮、按钮随机回应、闲置自动消息。
- 支持为每个互动按钮绑定一个上传素材，点击不同按钮展示不同素材。
- 支持上传一张形象图片，通过 aiflowlink 图片接口生成“大笑”拟人化素材。
- AI 生成会请求输出透明背景 PNG，并为对应按钮绑定轻量动效。
- 支持自由调节桌宠显示大小。
- 悬浮窗口会保留额外显示余量，避免首次打开或切换素材时顶部/底部被裁切。
- 所有配置和素材本地离线存储，设置修改后实时生效，无需重启。

## 运行方式

1. 安装 Node.js 18 或更高版本。
2. 在项目目录安装依赖：

```bash
npm install
```

3. 启动应用：

```bash
npm start
```

## 打包为 Windows 安装版

项目已内置 `electron-builder` 配置，可生成可安装的 `.exe` 安装包。

1. 安装或更新依赖：

```bash
npm install
```

2. 生成 Windows 安装包：

```bash
npm run dist
```

3. 构建完成后，在 `release/` 目录中找到安装程序：

```text
release/桌面宠物-Setup-1.0.0.exe
```

安装程序支持选择安装目录，并会创建桌面快捷方式和开始菜单快捷方式。

如果只想生成免安装目录版，用于本机快速测试：

```bash
npm run pack
```

## 使用说明

- 启动后会出现透明桌宠窗口。
- 首次没有素材时，会显示默认宠物占位形象。
- 悬停桌宠后点击“面板”，或右键桌宠，可以打开控制面板。
- 在控制面板点击“上传素材”导入宠物素材。
- 点击素材卡片的“切换”可立即替换桌宠显示内容。
- 点击“使用默认占位宠物”可切回默认形象。
- 拖动桌宠可改变位置，松开鼠标后自动保存。
- 调整“显示大小”滑块会实时改变悬浮窗口尺寸。
- 编辑单击回应、闲置消息、互动按钮和按钮绑定素材后，点击“保存设置”实时生效。

## AI 拟人化素材生成

控制面板的“AI 拟人化素材生成”可以上传一张 `png`、`jpg`、`jpeg` 或 `webp` 形象图片，并先生成一个“大笑”拟人化透明抠图素材。

生成完成后，应用会把透明背景图片保存到本地素材库，并自动创建或更新“大笑”互动按钮的素材绑定和轻量动效。

使用前需要准备 API Key，可在控制面板填写，也可以设置系统环境变量：

```powershell
$env:OPENAI_API_KEY="你的 API Key"
```

注意：AI 生成需要联网调用 `https://aiflowlink.top/v1/images/edits`；普通素材上传、切换和互动仍然是本地离线功能。

API Key 只用于发起本次生成请求，不会写入项目源码、安装包或本地配置文件。请不要把自己的 Key 写进 `main.js`、`panel.html`、`README.md` 等项目文件。

## 本地存储位置

应用会把配置和上传素材存放在 Electron 的 `userData` 目录中。Windows 通常位于：

```text
%APPDATA%\desktop-pet\
```

其中：

- `config.json`：保存素材列表、当前选中素材、窗口位置、尺寸和互动配置。
- `assets/`：保存上传后的本地素材副本。

也可以在控制面板点击“打开素材目录”直接查看素材文件夹。

## 注意事项

- `mov` 是否能播放取决于当前 Electron / Chromium 对该编码格式的支持；建议使用 H.264 编码的 `mp4` 或 `webm` 获得最佳兼容性。
- 如果桌宠窗口不明显，可在控制面板调大显示大小。
- 当前实现为纯本地应用，不上传素材或配置到网络。
